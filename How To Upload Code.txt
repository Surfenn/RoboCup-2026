SETUP INSTRUCTIONS (ARDUINO + TEENSY)

1. Install Required Software

• Install Arduino IDE
Download and install the Arduino IDE.

• Install Teensy Board Support (Teensyduino)
Download and install Teensyduino. This adds Teensy boards to the Arduino IDE.

2. After installing:

Open Arduino IDE

Go to Tools → Board

Select “Teensy 4.1” (or your correct Teensy board)

Install Required Libraries

Install the following libraries in Arduino (use Library Manager or place them inside Documents/Arduino/libraries):

Adafruit_BNO055
Adafruit_BusIO
Adafruit_MCP3008
Adafruit_MCP23017
Adafruit_TCS34725
Adafruit_Unified_Sensor
Pixy2 * Needs to be externally installed. File is within the Libraries Folder

After installing all libraries, restart the Arduino IDE.

Running the Code

• Open ONLY the main .ino file
• Connect the Teensy
• Select the correct board (Tools → Board → Teensy 4.1)
• Click Upload

You only need to run the main file.

***IMPORTANT***

If you edit any component file (.cpp or .h):

YOU MUST SAVE THE FILE before uploading again.

If you do not save, Arduino will upload the old version of the code.
